# CAMARA (camara)
CAMARA is a Linux Foundation project developing standardized telecom APIs as part of the Telco Global API Alliance. Supported by AT&T, Deutsche Telekom, Ericsson, Google Cloud, Microsoft, Nokia, and Vodafone, it enables developers to access operator capabilities through a unified, global API framework.
